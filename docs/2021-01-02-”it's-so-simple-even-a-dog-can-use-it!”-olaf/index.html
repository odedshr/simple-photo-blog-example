Adding text to a post is also very easy. Simply add a text file (with the extension .txt) to the folder and run the upload to compile the new post.Similarly you can add HTML files and MarkDown (.md) files if you want more control of the text's style.

Adding videos is also easy - you can either place your videos in the folder like you did with the photos;
Or if you have the file on a different website (such as vimeo or youtube) you can paste the link in a file with the name ending with ".video.txt" and the video will appear in the post.

The alphabetical order of the files makes a difference as this is the order of the different elements (photos, text and videos) will appear.Photo's filename is used as their ALTernative text, however if the name starts with a number (that is used for ordering) it will be omitted from the image's description.
